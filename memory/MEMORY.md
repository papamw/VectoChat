## Rappel rapide

- **Projet** : Application RAG (chat sur documents) — FastAPI + React + ChromaDB + JSON + OCR
- **Backend** : `backend/main.py` — un seul fichier, tous les endpoints
- **Frontend** : `frontend/src/` — React, composants dans `components/`, vues dans `views/`
- **Venv Python** : `/Users/papamamadouwade/AppBv1/venv/bin/python`
- **Démarrage** : `./start.command` ou voir CLAUDE.md pour les commandes détaillées
- **Build frontend** : `cd frontend && npm run build` (obligatoire après chaque modif)
